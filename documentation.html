Movie API Documentation

Introduction

Welcome to the MovieFlix API! This API provides access to information about movies, including details about directors, genres, release years, and more. Developers can use this API to build applications that display movie information, create user favorites lists, and more.

The API follows RESTful principles and returns data in JSON format.

Getting Started

To use this API, you'll need to:

  1. Make HTTP requests to the appropriate endpoints
  2. Handle the JSON responses in your application

Base URL

http://localhost:8080/

API Endpoints

Get All Movies

Endpoint: GET /movies

Description: Returns a list of all movies in the database.

Get Movie by Title

Endpoint: GET /movies/:title

Description: Returns data about a single movie by title.

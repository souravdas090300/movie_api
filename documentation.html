Movie API Documentation

Introduction

Welcome to the Movie API! This API provides access to a collection of movies, including details about directors, genres, and release years. Developers can use this API to retrieve movie information, user data, and create personalized lists of favorite movies.

Getting Started

To use this API, you'll need to make HTTP requests to the various endpoints listed below. All responses will be in JSON format.

API Endpoints

Get All Movies

Endpoint: GET /movies

Description: Returns a list of all movies in the database.

Get Movie by Title

Endpoint: GET /movies/:title

Description: Returns data about a single movie by title.

Future Updates

This documentation will be updated as new features are added to the API. Planned features include user authentication, personalized movie recommendations, and the ability to add reviews.
